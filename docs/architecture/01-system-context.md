## 外部实体

### 用户角色

| 角色 | 描述 | 主要交互 |
|------|------|----------|
| 运维工程师 | 日常运维和监控 | 日志检索、告警管理、仪表板 |
| 开发人员 | 应用调试和问题排查 | 日志检索、链路追踪 |
| 安全审计员 | 安全合规审计 | 审计日志、安全报表 |
| 系统管理员 | 系统配置和用户管理 | 系统设置、用户管理 |

### 外部系统

| 系统 | 集成方式 | 数据流向 |
|------|----------|----------|
| 应用服务 | Log Agent | 入站 |
| 告警通知渠道 | Webhook/API | 出站 |
| 身份提供商 | OIDC/SAML | 双向 |
| 对象存储 | S3 API | 出站 |
