# 安全架构

## 概述

本文档描述 NexusLog 系统的安全架构，包括认证、授权、数据安全和审计机制。

## 安全分层

```
┌─────────────────────────────────────────────────────────────────┐
│                         边界安全                                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  WAF / DDoS 防护 / TLS 终止                              │    │
│  └─────────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────────┤
│                         身份认证                                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  Keycloak (OIDC/SAML) / JWT 验证                         │    │
│  └─────────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────────┤
│                         访问控制                                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  OPA (RBAC/ABAC) / API 网关策略                          │    │
│  └─────────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────────┤
│                         数据安全                                 │
│  ┌─────────────────────────────────────────────────────────┐    │

└─────────────────────────────────────────────────────────────────┘
```

## 认证机制

### 支持的认证方式

| 方式 | 适用场景 | 实现组件 |
|------|----------|----------|
| OIDC | Web 用户登录 | Keycloak |
| SAML | 企业 SSO 集成 | Keycloak |
| API Key | 服务间调用 | API Gateway |
| mTLS | 内部服务通信 | Istio |

### JWT Token 结构

```json
{
  "sub": "user-id",
  "iss": "keycloak",
  "aud": "nexuslog",
  "exp": 1234567890,
  "roles": ["admin", "viewer"],
  "tenant": "default"
}
```

## 授权模型

### RBAC 角色定义

| 角色 | 权限 |
|------|------|
| admin | 全部权限 |
| operator | 日志查询、告警管理、仪表板 |
| viewer | 只读访问 |
| auditor | 审计日志访问 |

### OPA 策略示例

```rego
package nexuslog.authz

default allow = false

allow {
    input.user.roles[_] == "admin"
}

allow {
    input.user.roles[_] == "viewer"
    input.action == "read"
}
```

## 数据安全

### 敏感数据处理

| 数据类型 | 处理方式 |
|----------|----------|
| 密码 | 不存储明文，使用 bcrypt 哈希 |
| API Key | Vault 加密存储 |
| 日志中的 PII | 采集时脱敏 |
| 传输数据 | TLS 1.3 加密 |

## 审计日志

### 审计事件类型

| 事件类型 | 记录内容 |
|----------|----------|
| 登录/登出 | 用户、时间、IP、结果 |
| 配置变更 | 操作者、变更内容、时间 |
| 数据访问 | 查询者、查询条件、结果数量 |
| 权限变更 | 操作者、目标用户、变更内容 |
