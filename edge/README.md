# Edge - 边缘计算域（可选）

NexusLog 边缘计算相关模块，用于边缘节点的日志采集和本地处理。

## 目录结构

```
edge/
├── mqtt/       # MQTT 5.0 消息协议配置
├── sqlite/     # SQLite 本地存储配置
└── boltdb/     # BoltDB 嵌入式存储配置
```

## 技术栈

| 组件 | 审批级别 | 配置策略 |
|------|----------|----------|
| MQTT 5.0 | 常规 | 规则热更；分批升级 |
| SQLite | 常规 | 本地存储 |
| BoltDB | 常规 | 嵌入式 KV 存储 |
